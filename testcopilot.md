Create a GitHub access request topic for new employees and developers.

Trigger this topic when users ask:
- I need GitHub access
- Request GitHub access
- Give me GitHub permissions
- Repository access
- I cannot access GitHub

The topic should:

- Welcome the user.
- Ask for Full Name.
- Ask for Employee ID.
- Ask for Email Address.
- Ask for Project Name.
- Ask for GitHub Repository Name.
- Ask what level of access is needed (Read, Write, Admin).
- Ask why the access is required.
- Display a summary of all collected information.
- Ask the user to confirm the details.
- If confirmed, respond that the GitHub access request is ready for submission to the DevOps team.
- If the user says No, restart the information collection process.

Use friendly professional language and validate that required information is collected before proceeding.
